What is a work term report?

My work term reports are meant to track my experiences at each of my co-op placements. They will have all sorts of information, such as:

  • The company that I was working for
  • What my job was
  • What that company does
  • Who their clients are
  • What the work environment was like
  • Who I was working with
  • Which technologies and tools I used
  • My learning goals for that work term

About This Site

This website was created with Angular. I decided to make my own website instead of using a service like Wordpress because I thought it would be a great learning experience. Angular is currently one of the most popular web development technologies. It uses a combination of HTML, SCSS, and Typescript. I used bootstrap for the HTML side of the website because it provides some great components that would be extremely difficult to make from scratch.

Angular was the first web technology that I ever used in a workplace, and I have found that it was very easy to pick up and understand. Typescript is a strongly-typed superset of javascript, so it is more structured and can avoid many type issues that arise when using javascript.

(Github also has free website hosting for Angular, which is nice)
